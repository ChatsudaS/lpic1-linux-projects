# Project 2: System Information Report Generator

Bash script that collects system information and generates a timestamped report.

## How to Use


./system_report.sh


Creates a report file: `system_report_YYYYMMDD_HHMMSS.txt`

## What It Collects

- Hostname and uptime
- OS and kernel info
- CPU information
- Memory usage
- Disk space
- Top processes by CPU usage
- Network interfaces
