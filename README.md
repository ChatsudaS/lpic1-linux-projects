# LPIC-1 Linux Projects

A collection of bash scripts built while studying for the LPIC-1 Linux certification.

## Projects

### Project 1 — File Organizer
Automatically sorts files into folders by type (images, documents, videos)

### Project 2 — System Report
Generates a system health report showing CPU, memory, disk usage

### Project 3 — Automated Backup
Compresses important files and saves timestamped backups locally

### Project 4 — Log Analyzer
Scans log files and reports errors and warnings

## Skills Demonstrated
- Bash scripting
- File system management
- System monitoring
- Log analysis
- Linux automation
